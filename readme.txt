Requires the LUFA USB library http://www.fourwalledcubicle.com/LUFA.php
compiled with Lufa build 110528

The makefile will have to be modified to point to the root directory of the LUFA library on the build system.



This project is designed to compile and run on a Teensy 2.0++ (AT90USB1286) running at 16MHz
Any other configuration will require changes to the pin connections and the makefile
